## [0.1.2] - 2024-12-XX

### Добавлено
- **Фильтр лайков в топике**: Добавлена кнопка "<3" для просмотра всех лайков в топике
  - Кнопка отображается слева от ленты постов в топике
  - При клике собираются все лайки со всех страниц топика (параллельная загрузка по 5 страниц)
  - Отображение списка лайков с информацией: кто поставил лайк, кому, на какой пост, время и страница
  - Список отсортирован по времени (сначала самые новые)
  - Клик по элементу списка переходит к соответствующему посту
  - **Два фильтра для сортировки лайков**:
    - "Лайки поставлен:" - фильтр по пользователю, который поставил лайк
    - "Лайк получен:" - фильтр по пользователю, который получил лайк
  - Фильтры работают совместно (AND логика)
  - В каждом фильтре есть опция "Все" для показа всех лайков
  - Заголовок модального окна показывает количество отфильтрованных лайков
  - Кнопка остается видимой после навигации между страницами топика

### Исправлено
- Улучшена видимость кнопки просмотра лайков после переключения страниц в топике
- Исправлено отображение выпадающих списков фильтров (убраны фоновые изображения)

## [0.1.1] - Предыдущая версия
